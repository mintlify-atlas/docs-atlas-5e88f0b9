## Get All Users

<api-endpoint>
  <api-endpoint-request>
    <api-endpoint-request-method>GET</api-endpoint-request-method>
    <api-endpoint-request-url>/api/users</api-endpoint-request-url>
  </api-endpoint-request>
</api-endpoint>

Retrieve all user accounts with pagination and filtering.

<Note>
This endpoint requires `ROLE_ADMIN` permissions.
</Note>

### Query Parameters

<ParamField query="page" type="integer" default="0">
  Page number (zero-based)
</ParamField>

<ParamField query="size" type="integer" default="20">
  Number of items per page
</ParamField>

<ParamField query="sort" type="string">
  Sort field and direction (e.g., `login,asc`)
</ParamField>

<ParamField query="login" type="string">
  Filter users by login name (partial match)
</ParamField>

### Response

Returns an array of user objects.

<ResponseField name="id" type="integer">
  User ID
</ResponseField>

<ResponseField name="login" type="string">
  Username
</ResponseField>

<ResponseField name="firstName" type="string">
  User's first name
</ResponseField>

<ResponseField name="lastName" type="string">
  User's last name
</ResponseField>

<ResponseField name="email" type="string">
  User's email address
</ResponseField>

<ResponseField name="activated" type="boolean">
  Whether the user account is activated
</ResponseField>

<ResponseField name="langKey" type="string">
  User's language preference
</ResponseField>

<ResponseField name="authorities" type="array">
  Array of role names assigned to the user
</ResponseField>

<ResponseField name="imageUrl" type="string">
  URL to user's profile image
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://your-utmstack-instance.com/api/users?page=0&size=20" \
  -H "Authorization: Bearer eyJhbGciOiJIUzUxMiJ9..."
```

```python Python
import requests

url = "https://your-utmstack-instance.com/api/users"
headers = {"Authorization": "Bearer YOUR_TOKEN"}
params = {"page": 0, "size": 20}

response = requests.get(url, headers=headers, params=params)
users = response.json()
for user in users:
    print(f"{user['login']} - {user['email']} - Roles: {', '.join(user['authorities'])}")
```
</RequestExample>

## Get User by Login

Retrieve a specific user by their login name.

<Note>

## Filter Users by Login

<api-endpoint>
  <api-endpoint-request>
    <api-endpoint-request-method>GET</api-endpoint-request-method>
    <api-endpoint-request-url>/api/users/filter/{login}</api-endpoint-request-url>
  </api-endpoint-request>
</api-endpoint>

Search for users by partial login match, excluding the current user.

<Note>
This endpoint requires `ROLE_ADMIN` permissions.
</Note>

### Path Parameters

<ParamField path="login" type="string" required>
  Partial login name to search for
</ParamField>

### Query Parameters

<ParamField query="page" type="integer" default="0">
  Page number
</ParamField>

<ParamField query="size" type="integer" default="20">
  Page size
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://your-utmstack-instance.com/api/users/filter/analyst?page=0&size=20" \
  -H "Authorization: Bearer eyJhbGciOiJIUzUxMiJ9..."
```

```python Python
import requests

url = "https://your-utmstack-instance.com/api/users/filter/analyst"
headers = {"Authorization": "Bearer YOUR_TOKEN"}
params = {"page": 0, "size": 20}

response = requests.get(url, headers=headers, params=params)
users = response.json()
print(f"Found {len(users)} matching users")
```
</RequestExample>

<ResponseExample>

## Get User Authorities

<api-endpoint>
  <api-endpoint-request>
    <api-endpoint-request-method>GET</api-endpoint-request-method>
    <api-endpoint-request-url>/api/users/authorities</api-endpoint-request-url>
  </api-endpoint-request>
</api-endpoint>

Get a list of all available user roles/authorities.

<Note>
This endpoint requires `ROLE_ADMIN` permissions.
</Note>

<RequestExample>
```bash cURL
curl -X GET https://your-utmstack-instance.com/api/users/authorities \
  -H "Authorization: Bearer eyJhbGciOiJIUzUxMiJ9..."
```

```python Python
import requests

url = "https://your-utmstack-instance.com/api/users/authorities"
headers = {"Authorization": "Bearer YOUR_TOKEN"}

response = requests.get(url, headers=headers)
roles = response.json()
print("Available roles:", roles)
```
</RequestExample>

<ResponseExample>
```json 200 Success
[
  "ROLE_ADMIN",
  "ROLE_USER",
  "ROLE_ANALYST",
  "ROLE_VIEWER"
]
```
</ResponseExample>

## Response Headers

Paginated endpoints include:

- `X-Total-Count` - Total number of users
- `Link` - Pagination links (first, last, next, prev)

## User Roles

Common user authorities:

- `ROLE_ADMIN` - Full system access
- `ROLE_USER` - Standard user access
- `ROLE_ANALYST` - Security analyst permissions
- `ROLE_VIEWER` - Read-only access
