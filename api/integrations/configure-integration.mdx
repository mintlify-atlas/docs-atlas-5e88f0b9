## Create Integration

<api-endpoint>
  <api-endpoint-request>
    <api-endpoint-request-method>POST</api-endpoint-request-method>
    <api-endpoint-request-url>/api/utm-integrations</api-endpoint-request-url>
  </api-endpoint-request>
</api-endpoint>

Create a new integration.

### Request Body

<ParamField body="name" type="string" required>
  Integration display name
</ParamField>

<ParamField body="shortName" type="string" required>
  Short identifier for the integration
</ParamField>

<ParamField body="description" type="string">
  Description of what the integration does
</ParamField>

<ParamField body="integrationType" type="string" required>
  Type: COLLECTOR, FORWARDER, SCANNER, CLOUD, WEBHOOK, API
</ParamField>

<ParamField body="active" type="boolean" default="true">
  Whether integration is active
</ParamField>

<RequestExample>
```bash cURL
curl -X POST https://your-utmstack-instance.com/api/utm-integrations \
  -H "Authorization: Bearer eyJhbGciOiJIUzUxMiJ9..." \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Office 365 Logs",
    "shortName": "office365",
    "description": "Collects Office 365 audit logs",
    "integrationType": "CLOUD",
    "active": true
  }'
```

```python Python
import requests

url = "https://your-utmstack-instance.com/api/utm-integrations"
headers = {
    "Authorization": "Bearer YOUR_TOKEN",
    "Content-Type": "application/json"
}
data = {
    "name": "Office 365 Logs",
    "shortName": "office365",
    "description": "Collects Office 365 audit logs",
    "integrationType": "CLOUD",
    "active": True
}

response = requests.post(url, headers=headers, json=data)
integration = response.json()
print(f"Created integration ID: {integration['id']}")
```
</RequestExample>

<ResponseExample>
```json 201 Created
{
  "id": 15,
  "name": "Office 365 Logs",
  "shortName": "office365",
  "description": "Collects Office 365 audit logs",
  "integrationType": "CLOUD",
  "active": true,
  "module": null
}
```

```json 400 Bad Request
{
  "status": 400,
  "error": "Bad Request",
  "message": "A new utmIntegration cannot already have an ID"
}
```
</ResponseExample>

## Update Integration Configuration

<api-endpoint>
  <api-endpoint-request>
    <api-endpoint-request-method>PUT</api-endpoint-request-method>
    <api-endpoint-request-url>/api/utm-integration-confs</api-endpoint-request-url>
  </api-endpoint-request>
</api-endpoint>

Update an integration configuration parameter. This will mark the associated module as needing a restart.

### Request Body

<ParamField body="id" type="integer" required>
  Configuration ID to update
</ParamField>

<ParamField body="integration" type="object" required>
  Integration object with ID
</ParamField>

<ParamField body="confName" type="string" required>
  Parameter name
</ParamField>

<ParamField body="confValue" type="string" required>
  New parameter value
</ParamField>

<ParamField body="confDescription" type="string">
  Description
</ParamField>

<RequestExample>
```bash cURL
curl -X PUT https://your-utmstack-instance.com/api/utm-integration-confs \
  -H "Authorization: Bearer eyJhbGciOiJIUzUxMiJ9..." \
  -H "Content-Type: application/json" \
  -d '{
    "id": 50,
    "integration": {"id": 15},
    "confName": "tenant_id",
    "confValue": "new-tenant-id",
    "confDescription": "Updated Microsoft 365 Tenant ID"
  }'
```
</RequestExample>

<ResponseExample>
```json 200 Success
{
  "id": 50,
  "integration": {
    "id": 15,
    "name": "Office 365 Logs",

    }
  },

  "confValue": "new-tenant-id",
  "confDescription": "Updated Microsoft 365 Tenant ID"
}
```
</ResponseExample>

## Complete Integration Setup Example

```python Python
import requests

def setup_integration(api_url, token, integration_data, configs):
    """
    Create an integration and configure its parameters.
    """
    headers = {
        "Authorization": f"Bearer {token}",
        "Content-Type": "application/json"
    }
    
    # Step 1: Create integration
    response = requests.post(
        f"{api_url}/utm-integrations",
        headers=headers,
        json=integration_data
    )
    
    if response.status_code != 201:
        return {"success": False, "error": "Failed to create integration"}
    
    integration = response.json()
    integration_id = integration["id"]
    print(f"Created integration ID: {integration_id}")
    
    # Step 2: Add configuration parameters
    created_configs = []
    for config in configs:
        config_data = {
            "integration": {"id": integration_id},
            "confName": config["name"],
            "confValue": config["value"],
            "confDescription": config.get("description", "")
        }
        
        response = requests.post(
            f"{api_url}/utm-integration-confs",
            headers=headers,
            json=config_data
        )
        
        if response.status_code == 201:
            created_configs.append(response.json())
            print(f"  + Added config: {config['name']}")
        else:
            print(f"  - Failed to add config: {config['name']}")
    
    return {
        "success": True,
        "integration": integration,
        "configs": created_configs
    }

# Example: Setup Office 365 integration
integration_data = {
    "name": "Office 365 Logs",
    "shortName": "office365",
    "description": "Collects Office 365 audit logs",
    "integrationType": "CLOUD",
    "active": True
}

configs = [
    {
        "name": "tenant_id",
        "value": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
        "description": "Microsoft 365 Tenant ID"
    },
    {
        "name": "client_id",
        "value": "yyyyyyyy-yyyy-yyyy-yyyy-yyyyyyyyyyyy",
        "description": "Application Client ID"
    },
    {
        "name": "client_secret",
        "value": "secret-value-here",
        "description": "Application Client Secret"
    }
]

result = setup_integration(
    "https://your-utmstack-instance.com/api",
    "YOUR_TOKEN",
    integration_data,
    configs
)

if result["success"]:
    print(f"\nIntegration setup complete!")
    print(f"Integration ID: {result['integration']['id']}")
    print(f"Configs created: {len(result['configs'])}")
```
