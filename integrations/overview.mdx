## Integration Architecture

All UTMStack integrations use a plugin-based architecture:

- **GRPC Communication**: Plugins communicate with the UTMStack processing server via GRPC over Unix sockets
- **Real-time Processing**: Events are processed and enriched in real-time
- **Automated Configuration**: Plugins automatically sync configuration from the UTMStack server
- **Resilient Design**: Built-in retry mechanisms and error handling ensure reliable data collection
